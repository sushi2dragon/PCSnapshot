# Feature — Restore Engine

## Goal

Reconstruct a saved snapshot.

---

## Steps

1. Read snapshot file
2. Validate data
3. Build restore plan
4. Launch processes
5. Wait for windows
6. restore positions
7. apply context hints
8. run fallback macros

---

## Restore Order

1. background tools
2. terminals
3. IDEs
4. browsers
5. foreground app

---

## Restore Types

- full restore
- partial restore
- failed restore

---

## Output

- success message
- list of failed items
