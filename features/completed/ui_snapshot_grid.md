# Feature — Snapshot Grid UI

## Goal

Minimal UI for interacting with snapshots.

---

## Empty State

- centered "Take Snapshot" button

---

## After First Snapshot

- grid layout appears
- button moves to top-right

---

## Snapshot Tile

Contains:
- thumbnail
- name

---

## Hover Behavior

- dark overlay appears
- accent-colored "Restore" button appears

---

## Click Behavior

- clicking tile triggers restore
- restore button also triggers restore

---

## Layout

- responsive grid
- clean spacing
- no clutter

---

## Design Rules

- dark theme only (#1c1c1f base, #252528 card, #2a2a2d tile)
- teal #4bbfc3 as the only accent color
- minimal text
